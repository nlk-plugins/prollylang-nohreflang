== Description ==

The Polylang plugin automatically prints <em>&lt;link rel="alternate" hreflang="...</em> meta tags for translated Pages. This plugin just disables that, in case you are manually adding that elsewhere or just don't want that output.

You can find the source code at the GitHub repository:
https://github.com/ninthlink/nlk-plugins/tree/master/prollylang_nohreflang


== Installation ==

1. Upload file to the `/wp-content/plugins/` directory
2. Activate the Prollylang No HREFLANG plugin through the 'Plugins' menu in the WordPress Admin Panel. Just set it, and forget it
